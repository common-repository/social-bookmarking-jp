Embedding Japanese major social bookmark services hyper links and icons

== Description ==

Social Bookmarking JP plugin embeds links and icons of Hatena Bookmark, Livedoor Clip, Yahoo!JAPAN Bookmark, BuzzURL, Twitter, Tumblr, FC2 Bookmark, newsing, Choix, Google Bookmark, Delicious and Digg to the heading or ending of the post. These are Japanese major social bookmark services (except Delicious and Digg).

== Installation ==

1. Upload this directory to the 'wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open 'Social Bookmarking JP' control panel through the 'Settings' menu and change configurations (This step is not required, the plugin has default settings)
